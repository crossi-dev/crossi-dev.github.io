Velora — where Claude, ChatGPT & Gemini run a real business

A chat-first, multi-agent commerce platform for businesses in Argentina. It exposes a production MCP server with ~58 tools covering the full commerce cycle — and any MCP client can operate a real business through it, with HMAC-authenticated, multi-tenant access. This is the actual product, live.

~58 tools, 14 packs

Fiscal invoicing (AFIP/ARCA), MercadoPago payments and payment links, logistics, catalog, sales, cash register, and WhatsApp messaging via the Meta Cloud API.

Multi-agent over A2A

A supervisor + specialized sub-agents communicating over the A2A protocol, with signed agent identities (Ed25519), orchestrated with Google ADK and Gemini.

Built for production

Next.js / TypeScript on Cloud Run with Postgres. Idempotent money-path mutations, per-tenant isolation, webhook signature verification, async task queues.
