Featured project

Velora — a production
AI-native commerce platform.

Production · Live somosvelora.com ↗

A chat-first commerce platform where AI agents run the entire business: register sales, issue invoices, dispatch shipments, collect payments — all from a single conversation.

The problem

Small businesses in Argentina run on a tangle of WhatsApp messages, spreadsheets, and manual steps. Every sale, invoice, shipment, and payment requires a person to coordinate across 4–5 disconnected systems.

What I built

A multi-agent architecture where the owner manages everything through conversation. Each intent routes to the right specialist agent, which executes end-to-end without human relay.

The MCP server is engine-agnostic: the same 58 tools are consumable by Claude, ChatGPT, Gemini, or any MCP-compatible client. Each agent carries a cryptographic Ed25519 identity for verified agent-to-agent communication over the A2A bus.
